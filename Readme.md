# TemperatureMonitor

A cross-language project for monitoring and processing temperature data, featuring both C and C++ components.

## Project Structure

- `c/src` &mdash; C source files
- `cpp/src` &mdash; C++ source files

## Building the Project

### Prerequisites

- GCC or Clang for C
- g++ or clang++ for C++
- Make

### Build C Components

```sh
cd c
make run
```

### Build C++ Components

```sh
cd cpp
make run
```
